## Setup & Configure BOT

### Linux
1. clone project repo
   ```
   git clone https://github.com/hokireceh/ink.git 
   cd ink-gm-bot
   ```
2. run
   ```
   npm install
   npm run setup
   ```
3. Configure your accounts
   ```
   nano accounts/accounts.js
   ```
4. to start the app run
   ```
   npm run start
   ```


## Update Bot

To update bot follow this step :
1. run
   ```
   git pull
   ```
   or
   ```
   git pull --rebase
   ```
   if error run
   ```
   git stash && git pull
   ```
2. run
   ```
   npm update
   ```
2. start the bot
